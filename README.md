## Proyecto en R: Simulación de activo financiero

# Objetivo principal
En este proyecto se desarrolla mediante el paquete markovchain de R un proceso de simulación de la variación de un activo financiero en tres estados, sube, se mantiene y baja, ligado a los estados de las noticias del mercado, en otros tres estados: positivo, neutro y negativo. Buscamos además generar distintos estadísticos relevantes para este problema, calculados utilizando las simulaciones, empezando por el tiempo  medio hasta caída del activo.
